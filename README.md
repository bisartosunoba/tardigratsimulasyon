# Tardigrat Simulasyonu
Tardigrat isimli hayvanın, Fortran dili üzerinde oluşturulmuş bir "Tun Modu" simulasyonu.
Simulasyondaki amaç bu hayvanın hangi şartlar altında Tun Modu'na girceği hakkında bilgi sahibi olmak ve deney yapmak.


## Simulasyondaki Bağımlı Değişkenler:
- Tun Modu
- Aktif Mod


## Simulasyondaki Bağımsız Değişkenler:
- Su yüzdesi
- Sıcaklık (celcius)


## Simulasyondaki Kontrol Edilen Değişkenler:
- Basınç
- Radyasyon
